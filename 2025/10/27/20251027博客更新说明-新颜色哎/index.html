🎨 渐变色详细解析

🌅 早晨时段（柔和清新）

  • 黎明 04-06：浅米白 → 香槟 → 桃色 → 玫瑰粉（4色）
  • 日出 06-08：金黄 → 浅杏 → 粉橙 → 玫瑰红（4色）
  • 清晨 08-10：柠檬黄 → 薄荷黄绿 → 薄荷绿 → 天蓝 → 淡薄荷（5色）

☀️ 白天时段（明亮活力）

  • 上午 10-12：淡青 → 天蓝 → 云蓝 → 云白（4色）
  • 中午 12-14：清薄荷 → 青绿 → 湖绿 → 湖蓝（4色）
  • 下午 14-16：翠绿 → 海绿 → 青绿 → 碧蓝（4色）

🌆 黄昏时段（温暖浪漫）

  • 傍晚 16-18：金黄 → 琥珀 → 杏橙 → 珊瑚粉（4色）
  • 黄昏 18-19：亮金 → 金橙 → 杏橙 → 珊瑚红（4色）
  • 日落 19-20：橙粉 → 玫瑰粉 → 紫粉 → 紫罗兰（4色）

🌙 夜晚时段（神秘梦幻）

  • 晚霞 20-22：淡粉 → 樱粉 → 玫瑰 → 深紫红（4色）
  • 深夜 22-02：蓝紫 → 靛紫 → 宝石紫 → 深紫（4色）
  • 凌晨 02-04：深蓝紫 → 紫罗兰 → 淡紫 → 浅蓝紫（4色）

✨ 设计理念

  1. 渐变层次丰富：所有时段都使用 4-5 种颜色，过渡更平滑自然
  2. 自然光线模拟：每个时段的颜色灵感来自真实的自然光线变化
  3. 明亮主调：即使是夜间也保持适度亮度，避免过于沉闷
  4. 平滑过渡动画：1.5秒 CSS 过渡确保换色流畅优雅
  5. 对比度优化：按钮和文字颜色始终与背景形成良好对比，确保可读性
  6. 情感共鸣设计：色彩搭配符合人们对不同时段的心理感受

🔧 技术实现

  • 时间来源：本地系统时间（已移除公网 API 避免 CORS 问题）
  • 更新频率：每秒更新时钟显示，每小时切换背景和按钮配色
  • 响应式设计：完美适配桌面端、平板和移动端
  • PJAX 支持：单页应用无缝切换，无刷新体验
  • 智能重试：元素加载失败时自动重试，确保颜色正确应用
  • DOM 监听：MutationObserver 监听动态元素，实时更新样式
  • 性能优化：防抖机制避免重复更新，减少 DOM 操作
