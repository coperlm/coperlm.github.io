数论小芝士

欧拉定理：https://oi-wiki.org/math/number-theory/fermat/

$phi = n-p-q+1$，故 $c=m^{phi+2} mod \ n$ ，由欧拉定理可得 $c^{phi}\equiv 1 \ mod n $

则 $c\equiv m^{2} mod \ n $

直接开根即可
